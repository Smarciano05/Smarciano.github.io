Objectif :

L'objectif du projet était de concevoir et de développer un quiz de veille technique interne pour l'entreprise Omnilog. Ce quiz visait à permettre aux collaborateurs de tester et de renforcer leurs connaissances en matière de technologies et d'outils utilisés au sein de l'entreprise.

Mes missions :

Mission 1 : Initialisation du projet

  • - Création du projet Symfony.
  • - Configuration de Docker pour l'environnement de développement.
  • - Installation et configuration de la base de données avec MySQL.

Mission 2 : Développement des entités et fonctionnalités

  • - Création des entités Question et Réponse
  • - Développement des fonctionnalités pour la gestion des questions
  • - Mise à jour de la base de données et des entités avec les nouvelles fonctionnalités.

Mission 3 : Développement du formulaire et de la logique

  • - Création de la classe de formulaire pour l’entité Question.
  • - Mise en place de l'affichage des questions et réponses dans le formulaire.

Outils Utilisés :

PHP/SYMFONY

PHP et SYMFONY m'ont servi à gérer la logique de l'application côté back-end.

HTML/CSS/JS

HTML et CSS et JS ont permis de structurer et de styliser l'interface utilisateur côté front-end.

MYSQL

J'ai utilisé MySQL pour stocker et gérer les données de l'application.

VISUAL STUDIO CODE

J'ai utilisé VS CODE pour écrire et organiser mon code de façon efficace. Il m'a permis d'avoir une interface fluide tout en bénéficiant de fonctionnalités avancées.

DOCKER

Docker m'a aidé à lancer phpMyAdmin pour gérer la base de données sans configuration complexe.

GITLAB

Avec GitHub, j'ai sauvegardé et partagé les versions du projet facilement.

Problèmes Rencontrés :

Problème de configuration de la base de données avec Docker :

    Nécessité de repartir de zéro en utilisant une image MySQL dans Docker.

Problème avec la soumission du formulaire :

    Une erreur est survenue concernant la propriété "réponses", qui devait être un tableau. Cela a empêché le bon traitement des données lors de la soumission du formulaire.

Problème de logique pour la comparaison de la bonne réponse :

    L'extraction et la comparaison de la réponse soumise par l'utilisateur avec la bonne réponse n'ont pas fonctionné, ce qui a empêché l'affichage du message approprié et l'enregistrement des résultats en base de données.
