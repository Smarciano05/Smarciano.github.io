VELIKO WEB

Information Projet :

  • Nom du projet: VELIKO
  • Catégorie: Développement sous web
  • Date: 2024-2025

Objectif :

Fournir aux utilisateurs des informations en temps réel sur la disponibilité des vélos, y compris les vélos à assistance électrique, dans les 1400 stations de Paris et Île-de-France, avec une carte interactive centrée sur Paris et la possibilité de localisation personnelle.

Mes missions :

Mission 1 : Intégration de la carte

  • - Intégrer une carte interactive dans l'application.
  • - Afficher les stations sur la carte avec des marqueurs
  • - Localiser l'utilisateur sur la carte en temps réel.

Mission 2 : Gestions des données des stations

  • - Récupérer les données des stations via une API
  • - Afficher le nom de la station et le nombre de vélos disponibles

Mission 3 : Gestions des utilisateurs et fonctionnalités

  • - Gestions Utilisateurs
  • - Authentification et RGPD
  • - Ajout des stations favoris
  • - Gestion de la réservation

Outils Utilisés :

PHP/SYMFONY

PHP et SYMFONY m'ont servi à gérer la logique de l'application côté back-end.

HTML/CSS/JS

HTML et CSS et JS ont permis de structurer et de styliser l'interface utilisateur côté front-end.

MYSQL

J'ai utilisé MySQL pour stocker et gérer les données de l'application.

PHP STORM

J'ai utilisé PHPSTORM pour écrire et organiser mon code de façon efficace. Il m'a permis d'avoir une interface fluide tout en bénéficiant de fonctionnalités avancées.

DOCKER

Docker m'a aidé à lancer phpMyAdmin pour gérer la base de données sans configuration complexe.

GITHUB

Avec GitHub, j'ai sauvegardé et partagé les versions du projet facilement.

Problèmes Rencontrés :

1. Problème d'affichage des stations
  • - Difficulté à afficher les stations lié à l'API
  • - Difficulté à afficher la map
2. Problème de verification du compte
  • - Difficulté d'envoie des emails
  • - Difficulté à rendre le compte inactif
3. Erreurs de syntaxe dans le code :
  • - Vérification des parenthèses et des points-virgules
  • - Vérification de la structure des blocs PHP et HTML
