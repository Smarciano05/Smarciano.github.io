VELIKO JAVA

Information Projet :

  • Nom du projet: VELIKO
  • Catégorie: Développement sous java
  • Date: 2023-2025

Objectif :

Développer une interface administrateur en Java permettant la gestion des utilisateurs, la visualisation du parc de vélos sur une carte interactive et l’affichage de statistiques détaillées sur les réservations.

Mes missions :

Mission 1 : Gestion des utilisateurs

  • - Administration des utilisateurs (CRUD)
  • - Supprimer et bloquer un utilisateur
  • - Forcer le changement de mdp d'un utilisateur

Mission 2 : Gestions du parc

  • - La visualisation d’une MAP
  • - Distinctions de couleurs entre les vélos
  • - Ajout marqueur

Mission 3 : Gestions des Statistiques

  • - Collecte et traitement des données
  • - Création des éléments visuels du Dashboard
  • - Affichage dynamique et interaction

Outils Utilisés :

JAVA

JAVA m'a servi à gérer la logique de l'application côté back-end

HTML/CSS/JS

HTML et CSS et JS ont permis de structurer et de styliser l'interface utilisateur côté front-end.

MYSQL

J'ai utilisé MySQL pour stocker et gérer les données de l'application.

PHP STORM

J'ai utilisé PHPSTORM pour écrire et organiser mon code de façon efficace. Il m'a permis d'avoir une interface fluide tout en bénéficiant de fonctionnalités avancées.

DOCKER

Docker m'a aidé à lancer phpMyAdmin pour gérer la base de données sans configuration complexe.

GITHUB

Avec GitHub, j'ai sauvegardé et partagé les versions du projet facilement.

Conclusion :

Ce projet m’a permis de renforcer mes compétences en gestion des utilisateurs, intégration de la carte interactive et création d’un Dashboard dynamique. J’ai appris à structurer une application de manière efficace, à gérer les données en temps réel et à optimiser l’interaction avec l’utilisateur.
